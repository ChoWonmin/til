# Input

- unity에서 input interface를 재공

## Kye Code
- Return : Enter
- Escape : esc
- LeftArrow : 왼쪽 방향키

## Button
- GetKeyDown()
- GetKey()
- GetKeyUp()

## Mouse
- param
    - 0 : 오른쪽
    - 1 : 왼쪽

- GetMouseButtonDown(0)
- GetMouseButton(1)
- GetMouseButtonUp(0)

## Axis
- param : setting에서 명칭 변경 가능
    - "Horizontal"
    - "Vertical"
- GetAxis("Horizontal")
- GetAxisRaw("Vertical")
